# 背景图片设置说明

## Hero区域背景

请将您提供的海边晾衣绳图片保存为以下文件名：
- 文件路径: `/public/images/backgrounds/clothesline-beach.jpg`
- 推荐尺寸: 1920x1080 或更高分辨率
- 支持格式: JPG, PNG, WEBP

## 如何替换背景

1. 将您的海边晾衣绳图片重命名为 `clothesline-beach.jpg`
2. 保存到 `public/images/backgrounds/` 目录
3. 刷新网页即可看到新背景

## 备注

如果您的图片文件不存在，系统会自动fallback到SVG背景。
